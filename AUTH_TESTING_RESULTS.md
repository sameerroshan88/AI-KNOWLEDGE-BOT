## 📊 **User Data Management**
- **Storage Location**: Supabase Auth + user_metadata
- **Data Stored on Signup**:
  - `email`: User's email address
  - `full_name`: User's full name (in metadata)
  - `password`: Hashed by Supabase
  - `id`: UUID generated by Supabase

- **Session Management**:
  - localStorage stores: `{ email, name, id }`
  - Used for quick dashboard access
  - Cleared on logout
